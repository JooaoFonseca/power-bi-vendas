# power-bi-vendas

Este projeto foi desenvolvido durante o Intensivão de Power BI da Hashtag Treinamentos, com o objetivo de demonstrar o processo completo de transformação de dados brutos em dashboards interativos para apoio à tomada de decisão.

O projeto percorre todas as etapas de um fluxo de Business Intelligence, desde a importação e tratamento dos dados até a criação de visualizações analíticas.

OBJETIVO DO PROJETO:

Construir um dashboard interativo em Power BI que permita analisar indicadores relevantes do conjunto de dados utilizado, facilitando a interpretação de informações e geração de insights.

Principais objetivos:

  Transformar dados brutos em informações estruturadas
  Aplicar ETL (Extract, Transform, Load)
  Criar modelagem de dados eficiente
  Desenvolver visualizações analíticas
  Criar um dashboard interativo e intuitivo

TECNOLOGIAS UTILIZADAS:

  Power BI Desktop
  Power Query
  DAX (Data Analysis Expressions)
  Modelagem de dados relacional
  Visualização de Dados
